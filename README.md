# coffee-shop-api
Coffee Shop Rest API
<br>
<br>
1.Requirements
- MySQL
- Node.js
- npm

2.Installation
```bash
git clone https://github.com/syauqeesy/coffee-shop-api.git
cd zwallet
yarn install
```

3.Configuration
```bash
cp env .env
```
Edit .env file
```
# Server
PORT=
BASE_PATH=
BASE_URL=

# Database
DB_HOST=
DB_USER=
DB_PASSWORD=
DB_NAME=

# Authentication
JWT_SECRET_TOKEN=


3. Run the server
```bash
npm run dev
or
npm run start
```
You're good to go!
